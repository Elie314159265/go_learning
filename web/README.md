# Web Frameworks

Go Webフレームワークを学ぶディレクトリ

## ディレクトリ構成

- **stdlib-http/**: 標準ライブラリのnet/http
- **echo/**: Echo Webフレームワーク
- **gin/**: Gin Webフレームワーク

## 学習順序の推奨

1. stdlib-http - 基礎を理解
2. echo または gin - 高レベルフレームワークを学習

## その他のフレームワーク

- **Fiber**: Express風のAPI
- **Chi**: 軽量ルーター
- **Beego**: フルスタックフレームワーク

# Standard Library

Go標準ライブラリを学ぶディレクトリ

## トピック

- **strings**: 文字列操作
- **strconv**: 文字列変換
- **fmt**: フォーマット入出力
- **io/ioutil**: ファイル操作
- **time**: 日付と時刻
- **encoding/json**: JSON処理
- **net/http**: HTTP クライアント/サーバー
- **os**: オペレーティングシステム機能
- **flag**: コマンドライン引数パース

## 実行方法

```bash
go run strings_demo.go
```
